#项目名称：随心租
一个全品类租赁平台，用于解决市面上租赁平台不全面不完善问题

#项目介绍：
本项目是基于 Vue3 开发的响应式租赁平台前端，包含用户登录、注册、首页展示、商品分类、轮播图、导航吸顶，个人主页，发布商品，消息模块等功能。采用现代化工程化开发方式，模块化结构清晰，适合作为租赁类平台的前端基础框架。

#技术栈：
Vue 3
Vite
Pinia
Vue Router
Axios
MockJS
Element Plus

#项目运行
yarn
yarn dev

#功能模块
用户登录 / 注册
首页展示与布局
轮播图
导航吸顶
商品分类面板
个人信息展示
请求统一封装
模拟数据接口

#项目亮点
使用 Vue3 + Vite 构建，开发体验流畅
采用 Pinia 进行全局状态管理
Axios 封装统一处理请求、响应与错误
Mock 模拟后端数据，支持独立开发
页面模块化拆分，结构清晰
